Объектно-ориентированная система для управления различными типами публикаций (книги, журналы, газеты). Проект демонстрирует принципы ООП: наследование, полиморфизм, абстракцию и инкапсуляцию.

## Функции
*   Добавление различных типов публикаций (книги, журналы, газеты) через единый интерфейс
*   Просмотр всего каталога публикаций с детальной информацией
*   Поиск публикаций по автору/издательству
*   Отслеживание общего количества публикаций через статический счётчик
*   Валидация данных при создании и изменении публикаций

## Структура проекта
*   `Printable` — интерфейс, определяющий метод `printDetails()` для вывода информации
*   `Publication` — абстрактный базовый класс для всех типов публикаций
*   `Book`, `Magazine`, `Newspaper` — конкретные классы-наследники, реализующие специфичные для каждого типа поля и поведение
*   `Library` — класс для управления коллекцией публикаций
*   `Main` — класс с консольным пользовательским интерфейсом
